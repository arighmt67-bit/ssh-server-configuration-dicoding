01 — Autentikasi

Key-based authentication

SSH key pair ed25519 dibuat di mesin pertama, public key disalin ke mesin kedua via ssh-copy-id. Autentikasi password dinonaktifkan sepenuhnya setelah key terkonfirmasi.

02 — Hardening

Pengamanan sshd_config

Port diganti dari 22 ke 2000, root login dinonaktifkan, hanya public key yang diizinkan. Password auth di-set no untuk mencegah brute-force.

03 — Dokumentasi

Log aktivitas SSH

Seluruh sesi SSH terekam di journalctl dan diekspor ke dua format: plain text untuk keterbacaan, dan JSON untuk keperluan parsing programatik.

04 — Otomatisasi

Shell script pembersih log

Script hapus-log.sh berjalan dalam loop while, menampilkan disk usage journal, vacuum hingga 10 MB, lalu tampilkan hasilnya kembali.

Konfigurasi SSH yang diterapkan
/etc/ssh/sshd_config
# Port default diganti untuk menghindari scan otomatis
Port                        2000

# Hanya izinkan autentikasi via public key
PubkeyAuthentication    yes
PasswordAuthentication  no

# Root tidak boleh login langsung via SSH
PermitRootLogin         no
